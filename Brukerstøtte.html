Sette opp AD (AD DS)

  • Steg 2
  • Steg 3
  • Steg 4

Steg 1: Logg inn på serveren og åpne Dashboard. Klikk deretter på "Add roles and features".

Steg 2: Trykk på Next til du kommer til Server Roles. Aktiver "Active Directory Domain Services", velg "Add Features", og trykk Next.

Steg 3: Det er ikke nødvendig å aktivere noe mer her, så trykk Next igjen.

Steg 4: Du får beskjed om at AD DS krever DNS Server. Dette installeres automatisk. Trykk Next.

Steg 5: Huk av for automatisk restart. Trykk Yes og deretter Install.

Steg 6: Vent til installasjonen er ferdig, og trykk Close.

Steg 7: Start serveren på nytt hvis den ikke gjør det automatisk.

Steg 8: Når serveren har startet, vil du se at AD DS vises i Dashboard. Trykk på AD DS.

Steg 9: Klikk på det gule varselet i Dashboard og velg "Promote this server to a domain controller".

Steg 10: Velg "Add a new forest" og skriv inn "ad.domain". Trykk Next.

Steg 11: Velg functional level (2025) og sett et restore passord. Trykk Next.

Steg 12: Trykk Next.

Steg 13: Sjekk at NetBIOS navnet stemmer. Trykk Next.

Steg 14: AD databasen lagres på standardplassering. Trykk Next.

Steg 15: Se gjennom oppsummeringen. Du kan også trykke "View script" for å se PowerShell-koden. Trykk Next.

Steg 16: Du får en DNS-advarsel. Dette er normalt. Trykk Install.

Steg 17: Serveren vil restarte automatisk.

Steg 18: Koble deg til serveren igjen etter omstart.

Steg 19: Gi serveren litt tid til å gjøre ferdig oppsettet.

Steg 20: Logg inn som Administrator. Du logger nå inn på domenet ditt.

Steg 21: AD DS og DNS skal nå vises som grønne i Dashboard.
